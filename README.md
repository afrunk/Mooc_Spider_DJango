# Visualization of MOOC education resources
## Part 1 Spyder
抓取MOOC网站的免费课程的如下四个分类
- JAVA  
- C\C++\C# 
- Python 
- 网络安全（信息安全）

具体抓取信息如下
- 课程封面
- 课程名
- 主讲人（老师）
- 累计参与课程人数
- 评分（或者星级）
- 课程链接
- 评论数
